## 服务器统一规划配置安装

一、创建文件夹

```shell
mkdir data
cd data
mkdir program
cd program
mkdir software
```

二、在software安装jdk

```shell
cd data/program/software

yum -y list java*

yum install -y java-1.8.0-openjdk-devel.x86_64
```

三、修改hosts

```shell
vim /etc/hosts

47.92.162.52 bigdata1
47.92.199.155 bigdata2
```

四、关闭防火墙

```shell
# 查看防火墙状态
service iptables status
# 关闭防火墙
service iptables stop
```
